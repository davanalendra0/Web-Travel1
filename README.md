<h2>🎓 Informasi Proyek</h2>
<p>Arctic Travel adalah sebuah landing page minimalis yang dirancang sebagai etalase visual untuk destinasi wisata musim dingin.</p>
<br>
<h2>🚀 Fitur dan Bagian Utama</h2>
<ul>
<li>Hero Section : Menghadirkan kesan pertama yang dingin dan luas melalui visual pemandangan Arktik yang ikonik.</li>
<li>Destinasi Pilihan : Bagian yang menampilkan beberapa lokasi ikonik dengan tata letak kartu yang sederhana namun terorganisir.</li>
<li>Navigasi Minimalis : Menu navigasi yang bersih di bagian atas untuk memudahkan pengguna menjelajahi konten tanpa gangguan visual.</li>
</ul>
<br>
<h2>⚙️ Teknologi yang Digunakan</h2>
<p>Proyek ini dibangun murni menggunakan teknologi dasar web untuk menjaga performa tetap ringan :</p>
<ul>
<li>HTML5 : Untuk struktur halaman dan penempatan elemen gambar.</li>
<li>CSS3 : Untuk styling tema minimalis, pengaturan kontras warna, dan tata letak responsif.</li>
<li>JavaScript : Untuk interaksi navigasi sederhana.</li>
<li>Deployment : Platform hosting yang digunakan adalah Vercel untuk penerapan cepat dan integrasi Git.</li>
</ul>
